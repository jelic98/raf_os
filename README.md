# RAF OS

Repozitorijum koji sadrži urađene domaće zadatke i projekte iz kursa "Operativni sistemi" na Računarskom fakultetu.

Modifikacije Linux kernela su označene komentarima ``` // DOMACI ``` ili ``` // PROJEKAT ```

Domaći 1: [tekst](https://bitbucket.org/jelic98/raf_os/src/domaci-1/domaci-1.pdf), [implementacija](https://bitbucket.org/jelic98/raf_os/src/domaci-1)

Projekat 1: [tekst](https://bitbucket.org/jelic98/raf_os/src/projekat-1/projekat-1.pdf), [implementacija](https://bitbucket.org/jelic98/raf_os/src/projekat-1)
